# EDNA Sales App Demo

A sales and business support web application demo built with JavaScript, React, HTML, and CSS.

## Project Overview

This project is a demo sales application designed to support business operations, product presentation, customer follow-up, and digital sales workflows.

The project demonstrates how a small business can use a simple web application to organize sales information and improve digital operations.

Private business/client information was removed or replaced with demo content for confidentiality.
